Keller Solutions Recipe Grabber

This is a proof-of-concept application demo, and as such is not authorized for commercial use. The application will grab recipies hosted on the myrecipes.com Web site. Simply copy a URL from the site and paste it below to grab the recipe.

examples:
http://www.myrecipes.com/recipe/slow-cooker-macaroni-beef-10000001875197/
http://www.myrecipes.com/recipe/honeyed-yogurt-mixed-berries-with-whole-grain-waffles-10000000671034/
http://www.myrecipes.com/recipe/toffee-french-toast-with-pecans-10000001842381/
